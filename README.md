# BuoyCalc iOS v3.4

SwiftUI-прототип iOS-приложения для предварительного расчёта буйковой постановки.

## Текущая архитектура

Проект описывает систему как последовательность элементов сверху вниз:

```text
Буй
↓
Соединитель
↓
Буйреп / линия
↓
Прибор / нагрузка
↓
Соединитель
↓
Линия
↓
Якорь
```

## Что есть в v3.4

1. расчётная модель;
2. буй;
3. пользовательские буи;
4. последовательность постановки;
5. участки швартовной линии;
6. пользовательские участки линии;
7. соединители;
8. пользовательские соединители;
9. приборы / нагрузки;
10. якорь;
11. пользовательские якоря;
12. грунт;
13. источники данных;
14. проверки системы;
15. текстовый отчёт;
16. markdown-экспорт;
17. PDF-экспорт;
18. 2D-схема сбоку;
19. визуальная цепочка постановки;
20. цепочка постановки в PDF.
